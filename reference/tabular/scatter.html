Scatter#

import hvplot.pandas  # noqa

scatter plots are a good first way to plot data with non continuous axes.

from bokeh.sampledata.iris import flowers as df

df.sample(n=5)
sepal_length sepal_width petal_length petal_width species
5 5.4 3.9 1.7 0.4 setosa
123 6.3 2.7 4.9 1.8 virginica
38 4.4 3.0 1.3 0.2 setosa
130 7.4 2.8 6.1 1.9 virginica
119 6.0 2.2 5.0 1.5 virginica 
df.hvplot.scatter(x='sepal_length', y='sepal_width', by='species', 
                  legend='top', height=400, width=400)

As for most other types of hvPlot plots, you can add fields to the hover display using the hover_cols argument. It can also take â€œallâ€ as input to show all fields.

df.hvplot.scatter(x='sepal_length', y='sepal_width', s='petal_length', scale=5, by='species', 
                  legend='top', height=400, width=600,
                  hover_cols=["species", "sepal_length", "sepal_width", "petal_width"])

You can add the â€˜sâ€™ parameter in scatter to specify the marker plot size and add the â€˜scaleâ€™ parameter to specify what the scaling factor should be.
